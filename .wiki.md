# Project Summary
The project is an advanced quiz application designed for educational institutions, focusing on integrity and security during online assessments. It incorporates AI-powered proctoring features, allowing real-time monitoring of students to prevent cheating. The application includes comprehensive admin controls for managing quiz attempts, student data, and performance analytics.

# Project Module Description
## New Features Added
1. **Admin Delete Functionality**
   - Admin can delete individual attempts and clear all history.
   - Students can re-attempt quizzes after deletion.

2. **Activity Logs Management**
   - All admin actions are logged with timestamps.
   - Admin can clear all activity logs.

3. **Enhanced Camera Features**
   - Draggable and resizable camera preview.
   - Constant monitoring for cheating (person and phone detection).

4. **AI-Powered Cheat Detection**
   - Detects multiple persons, cell phones, and absence of the student during the quiz.

# File Description Inventory
- **src/components/AdminDashboard.tsx**: Admin interface for managing quizzes and attempts.
- **src/components/AttemptsList.tsx**: Displays list of quiz attempts with options to view and delete.
- **src/components/DraggableCameraPreview.tsx**: Draggable camera interface for monitoring students.
- **src/components/Quiz.tsx**: Main quiz component handling the quiz logic and UI.
- **src/lib/db.ts**: Database operations for managing attempts and logs.
- **src/lib/objectDetection.ts**: AI object detection for cheating prevention.
- **src/hooks/use-toast.ts**: Custom hook for toast notifications.

# Usage
1. **Install Dependencies**:
   ```bash
   cd shadcn-ui
   pnpm install
   ```

2. **Build the Project**:
   ```bash
   pnpm run build
   ```

3. **Run the Application**:
   ```bash
   pnpm run start
        (or)
   pnpm run dev
   ```
